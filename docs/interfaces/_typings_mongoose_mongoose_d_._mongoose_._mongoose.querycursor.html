  • constructor(query: Query, options: Object): QueryCursor<T>

  • Overwrites Readable.constructor

    • Defined in typings/mongoose/mongoose.d.ts:607

    A QueryCursor is a concurrency primitive for processing query results one document at a time. A QueryCursor fulfills the Node.js streams3 API, in addition to several other mechanisms for loading documents from MongoDB one at a time. Unless you're an advanced user, do not instantiate this class directly. Use Query#cursor() instead.
